내가 오늘 배우고 느낀점

오늘은 메인 콘텐츠 두번째 그룹에 마지막 섹션인 새소식 섹션의 구조를 설계하고 디자인 입혀보는것 까지 진도를 나갔다. 구조를 짜는 것은 큰 힘 들이지 않고 해낼 수 있었는데 디자인에서 float 부분이 막혔었다. 옆에분과 상의를 해보면서 문제점을 파악할 수 있었고 해답까지 알아낼 수 있었다. 또 강사님이 똑같은 디자인을 4가지 다른 방법으로도 할 수 있다고 하셔서 혼자 자습할때 다시 시도해 볼 예정이다.

figure and figcaption

  • <figure></figure>은 이미지와 이미지를 설명해주는 <figcaption></figcaption>(설명글) 을 하나로 묶는데 사용되는 태그이다.
  • 사용법은 다음과 같다: <figcaption></figcaption>은 <img>를 설명해주는 설명글임으로 aria-labelledby라는 속성으로 서로를 명시적으로 묶어줄 수 있다.

ETC

  • flex는 단일 row, 단일 column을 컨트롤 할때 사용하고, grid는 다중 row, 다중 column을 컨트롤 할때 사용한다.
