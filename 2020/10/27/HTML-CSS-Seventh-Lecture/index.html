TIL 20/10/27 HTML/CSS 일곱번째 강의

Posted on 2020-10-27 In dev , writing

내가 오늘 배우고 느낀점

오늘은 메인 콘텐츠안에 두번째 그룹에 공지사항 및 자료실 섹션을 HTML 구조 설계와 CSS 디자인까지 해보았다. 내가 한 구조설계를 옆에분과 비교를 해보았는데 완전히 다른 구조가 나왔고 내 생각에는 내 구조가 더 비효율적인거 같아서 구조를 바꿨는데, 강사님이 두 구조를 비교하시면서 맞고 틀린건 없다고 하셨다. 매일 아침 한시간씩 일찍 오다보니까 오늘은 수업시간에 조금 졸아서 키워드만 듣고 설명을 못 들은 부분들이 있었다. 다시 구글에서 찾아보고 정리하는 시간을 오늘은 가져야겠다.

aria-labelledby

  • 한 요소가 다른 요소에 레이블 지정한다는 의미관계를 요소들 사이에 생성한다.
  • 예를들어 이런 코드가 있을때 맨아래에 있는 <a></a>태그가 최상단에 있는 <h2></h2>태그와 연결되어 있다는것을 명시적으로 보여준다.
  • 사용하려면 aria-labelledby값과 이어줄 다른요소의 id값이 일치해야된다.

text-overflow

  • 박스안에 text가 넘칠때 긴 글을 한줄로 나타내주고 나머지를 생략기호로 표현하고 싶을때 사용한다.
  • text-overflow를 구현하기 위해선 다음과 같은 조건이 필요하다:
    • overflow가 hidden, scroll, auto여야 한다
    • white-space: nowrap;으로 텍스트가 줄바꿈이 되지 않아야한다.
    • width 또는 height 값이 고정값으로 있어야한다.
  • text-overflow의 값으로는 다음이 있다:
    • clip: 기본값. 텍스트를 자른다.
    • ellipsis: 잘린 텍스트를 “…”으로 표현한다.
    • string도 있지만 지원하는 브라우저가 현재 없다.

ETC

  • role어트리뷰트는 <div>같이 의미 없는 태그에 의미 부여를 할때 사용한다. <h2></h2>와 같은 시맨틱 태그에는 role로 역할을 바꿔주는 행동은 지양해야한다.
  • <time></time>태그에 datetime이라는 어트리뷰트로 날짜 정보를 넣어줄 수 있다.
  • <script>요소가 외부 스크립트를 참조 할때 브라우저가 페이지의 파싱을 모두 끝내고, 즉 브라우저가 html요소들을 다 불러들인후 스크립트를 실행하려면 속성으로 defer를 주면 된다.

퀴즈에서 배운점

  • <pre></pre>태그는 태그안에 입력한 문장 형식 그대로를 브라우저에 표현한다. 공백, 줄바꿈 모두 표시되고 특수문자도 표시할 수 있다.
