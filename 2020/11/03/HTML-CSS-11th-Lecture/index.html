TIL 20/11/02 HTML/CSS 열한번째 강의

Posted on 2020-11-03 In dev , writing

내가 오늘 배우고 느낀점

어제부터 몸상태가 별로 였어서 수업시간에 집중을 잘 못했고 TIL도 어제 못적은걸 오늘 아침에 적는다. 처음으로 반응형 웹에 대해서 진도를 나갔는데, 반응형 웹을 잘 알기 전에는 크기 조정만 해주면 될줄 알아서 쉬운 작업일줄 알았는데, 막상 배워보니 고려해야될 부분들이 되게 많았다. 그리고 예전에 웹사이트를 학교에서 친구들과 만들어 봤을때 별 생각없이 처음에 데스크탑용 웹사이트를 만들어 놓고 거기서 모바일 용으로 줄여나가면서 반응형 웹을 구현했었는데, 모바일 first가 좋은 방법이라는것도 알았다. 몸 관리를 잘해서 얼른 나아서 수업에 더 집중을 해야겠다.

picture

  • <picture></picture>는 다른 디바이스나 화면 크기에 맞춰서 다른 사진들을 보여주게 할 수 있는 요소이다.
  • <picture></picture>는 하나이상의 source와 하나이상의 img를 갖는다.
  • <source>는 다른이미지들을 각각 참조하게 해주는 srcset이라는 속성과 이미지가 언제 가장 적합할때 쓰일까를 정해주는 media라는 속성을 가진다.
  • <img>요소는 <picture></picture>요소를 지원하지 않는 브라우저나 <source>요소가 모두 조건을 만족하지 못할때 사용된다.

ETC

  • svg 이미지는 확대 축소해도 해상도가 깨지지 않음.
  • svg 코드는 alt값을 줄 수 없어서 title속성으로 명시를 해줘야된다.
  • 반응형 디자인을 짤 때 모바일 버전부터 디자인을 시작 하는 것이 좋다.
